## 📌 About The Project

The **Employee Attendance Management System** is a simple web-based system designed to register and manage employee attendance records.

The system allows an employee to enter their **name** and **entry time** through a simple attendance form. After submitting the form, the information is sent to the PHP file and stored in a **MySQL database**.

The manager can then access the attendance records through a password-protected page and view all registered employees in an organized dashboard.

---

## 🎯 Project Purpose

The main purpose of this project is to demonstrate how a simple web application can be connected to a MySQL database to collect, store, and display employee attendance information.

It provides a basic example of how **HTML, CSS, PHP, and MySQL** work together to create a functional web-based system.

## 🔄 How The System Works

The system follows a simple process:

### 1. Employee Registration

The employee opens the attendance page and enters their name and entry time.

### 2. Submit Attendance

The employee clicks the **Submit Attendance** button.

### 3. Data Processing

The submitted information is sent to the PHP file, which processes the attendance data.

### 4. Database Storage

The PHP file connects to the MySQL database and stores the employee's attendance record.

### 5. Manager Access

The manager enters the correct password to access the attendance records.

### 6. Attendance Dashboard

The manager can view all stored records in a table containing the employee ID, name, entry time, and attendance status.

## 🗃️ Database Structure

The project uses a MySQL database to store employee attendance records.

| Field        | Description                               |
| ------------ | ----------------------------------------- |
| `id`         | Unique ID for each attendance record      |
